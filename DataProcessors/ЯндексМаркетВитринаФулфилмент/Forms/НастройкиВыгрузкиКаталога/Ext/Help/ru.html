Настройки выгрузки каталога

Форма предназначена для настройки опций по-умолчанию для выгрузки каталога в Яндекс.Маркет.

Чтобы указать вид цены базового прайс-листа, предварительно создайте вид цен для работы с Яндекс.Маркет и заполните прайс-лист. Информация из прайс-листа может быть использована для удобного формирования перечня публикаций (кнопка "Заполнить по базовому прайс-листу" в форме выгрузки каталога), а также в качестве источника информации для ориентировочный цен товара при получении рекомендуемых связей.

В поле Источник категории товара укажите из какого источника выгружать информацию о категории товара (вид номенклатуры, иерархия номенклатуры или товарная категория).
