Учетная политика финансового учета

Регистр служит для хранения настроек учетной политики финансового учета.
Каждая запись настроек имеет определенный период действия. Начало периода задается в форме записи настройки. Данная запись будет действовать до даты, предшествующей дате начала действия следующей записи настроек, или до конца текущего отчетного периода (года), если более поздних записей не введено. Ознакомиться с историей изменения настроек по организации можно при нажатии по гиперссылке История изменений.

Метод оценки стоимости товаров

Для выбора доступны следующие методы оценки:

  • Средняя (взвешенная) за месяц;
  • ФИФО (взвешенная оценка);
  • ФИФО (скользящая оценка).

Метод ФИФО (скользящая оценка) доступен для выбора в случае, если включено использование партионного учета (раздел НСИ и администрирование - Финансовый результат и контроллинг - Учет товаров)

Настраивается детализации дополнительных расходов по статьям. Если признак установлен, в отчетах "Валовая прибыль", "Себестоимость товаров" и "Дерево себестоимости продукции" можно увидеть детализацию дополнительных расходов до статей расходов.

Учет внеоборотных активов в управленческом учете
  • По стандартам международного учета - амортизация начисляется с даты документа, изменение параметров влияет на начисление амортизации с даты документа;
  • По ПБУ 6/01 и 14/2007 - амортизация начисляется со следующего месяца, изменение параметров влияет на начисление амортизации со следующего месяца.
  • Вид цены ТМЦ в эксплуатации - Используется при учете ТМЦ переданных в эксплуатацию (например, при распределении амортизации группового ОС между подразделениями, в которых находятся ТМЦ).

Учет дисконтированной кредиторской задолженности поставщикам


В случае включения учёта дисконтированной задолженности ТМЦ будут приниматься по себестоимости равной приведенной стоимости, а не номинальной. Приведенная стоимость будет рассчитываться в закрытии месяца исходя из ставки и срока дисконтирования, указанных в Договоре с контрагентом.

По гиперссылке Ставки дисконтирования можно указать ставку по умолчанию для каждой валюты взаиморасчетов, которая будет автоматически заполняться в договоры контрагентов. А в поле Срок для применения дисконтирования указывается количество календарных дней, влияющее на включение задолженности в разряд дисконтируемой. Если отсрочка платежа по задолженности превышает этот срок, то задолженность будет дисконтирована.

Статья списания процентных расходов и её Аналитика списания будут использоваться в регламентной операции закрытия месяца "Расчет процентных расходов дисконтирования".
