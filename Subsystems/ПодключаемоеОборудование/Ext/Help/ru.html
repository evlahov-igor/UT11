Подключаемое оборудование

Назначение подсистемы

Подсистема предназначена для настроки и подключения различного торгового обрудования: сканеров штрихкодов, терминалов сбора данных, Контрольно-кассовых машин (ККМ), работающих в различных режимах и т.д.

Настройка подсистемы

Для настройки торгового оборудования в разделе Администрирование требуется установка следующих опций:

  • Для возможности настройки обменов с подключаемым оборудованием для осуществления розничных продаж в разделе Продажи требуется установить функциональную опцию Розничные продажи.
  • Для возможности использования различного внешнего оборудования для осуществления розничных продаж в разделе Настройки РМК и оборудования требуется установить функциональную опцию Использовать подключаемое оборудование.

Работа подсистемы

Для возможности осуществления розничных продаж необходимо заполнить справочник хранения информации о настройках рабочих мест кассиров (Настройки РМК).
В форме справочника необходимо указать в поле Рабочее место, то рабочее место кассира, которое необходимо настроить (по умолчанию в данном поле отображается текущий пользователь).
Настроить правила обмена при работе с кассами ККМ и электронными весами, которые работают в режиме OffLine. С помощью данных правил настраивается список товаров выгружаемых в ККМ и электронные весы, назначаются соответствующие коды для работы с подключаемыми устройствами в удаленном режиме.
Настроить шаблоны для используемых магнитных кодов карт лояльности.
