Распределение расходов

Документ предназначен для распределения постатейных расходов между партиями производства и направлениями деятельности. Для выполнения распределения в списке отражены нераспределенные расходы с детализацией по статьям расходов и аналитике расходов.

Для настройки распределения двойным щелчком необходимо вызвать окно настроек, либо выделить перечень строк в списке расходов и нажать кнопку "Сформировать документы распределения". Документ по умолчанию заполняется на основании правила распределения расходов заданного в статье расходов.

В форме задается правило распределения расходов между направлениями деятельности. Возможно распределение как в автоматическом режиме с применением базы распределения, так и вручную по указанным коэффициентам.

В поле Распределять на возможно выбрать способ распределения расхода:

  • Все направления деятельности по базе. Распределение будет выполнено между всеми направлениями деятельности в соотстветсвии с выбранной базой распределения.
  • Указанные направления деятельности по базе. Распределение будет выполнено между указанными направлениями деятельности в соотстветсвии с выбранной базой распределения. Указать направления деятельности, между которым необходимо выполнить распределение, возможно с помощью нажатия на гиперссылку "Указать" рядом с полем.
  • Направления деятельности вручную. Распределение выполняется в соответствии с заданными вручную коэффициентами.

В зависимости от выбранного способа распределения в документе меняется состав закладок документа.

Основное

Закладка содержит основные реквизиты документа Номер, Дата, Комментарий и пр.
Можно указать для какого вида учета осуществляется настройка правила распределения: для упр. учета (распределяются суммы упр. учета), для регл. учета (распределяются суммы регл. учета) или для нал. учета (распределяются суммы нал. учета).

Правило распределения

На закладке можно указать базу распределения, по которой будут определятся направления деятельности и их веса.

Доступные базы распределения:

  • Выручка от продаж. Базой распределения служит сумма выручки от продаж организации, распределенная по видам деятельности.
  • Себестоимость продаж. Базой распределения служит сумма себестоимости продаж, распределенная по видам деятельности, осуществляемым организацией.
  • Валовая прибыль. Базой распределения служит сумма выручки от продаж за минусом себестоимости продаж, распределенная по видам деятельности.

Направления деятельности

Содержит таблицу, в которой можно вручную указать, сколько долей стоимости нужно отнести на определенное направление деятельности.
