Персональные настройки

Монитор целевых показателей предназначен для контроля и анализа целевых показателей деятельности предприятия. Для анализа доступны не только значения показателей, но и степень достижения поставленных целей.

Особенности:

  • предварительно заполненный список из более чем 20 целевых показателей и вариантов их анализа,
  • возможность добавления новых показателей или изменения существующих при наличии прав доступа,
  • краткая и расширенная форма представления показателей с отображением таблиц и диаграмм,
  • группировка показателей по степени их важности или по категориям целей,
  • детализация информации о показателе и варианте его расчета.

Группировка показателей

Можно установить один их следующих вариантов группировки: по категориям целей предприятия или по состоянию.


По категориям целей. Все показатели в соответствии с реквизитом Категория целей справочника Структура целей.

По состоянию. Все показатели будут разбиты на следующие группы в соответствии с состоянием показателя:

  • Неприемлемое состояние,
  • Приемлемое состояние,
  • Цель достигнута,
  • Состояние не определено,
  • Рассчитано с ошибками.

Количество колонок

По умолчанию данные в мониторе выводятся в три колонки. При необходимости пользователь может установить любое количество колонок.

Размер диаграмм и таблиц

Изменение размера доступно со следующим шагом - 75%, 100 % (стандартно), 125%, 150%, 175%, 200% (крупный) и произвольный. Произвольный масштаб может быть установлен с шагом 1 в интервале от 75% до 500%.

Период автообновления

По умолчанию период обновления установлен, равным 15 минутам. Пользователь может установить любую длительность (в минутах) периода автоматического пересчета данных в мониторе показателей. При этом будут обновляться только те показатели, период актуальности которых истек.

Варианты анализа

Настройка списка вариантов анализа, которые будут выводиться в монитор целевых показателей, определяется на странице Варианты анализа.

По умолчанию все доступные пользователю варианты анализа выводятся в виде диаграмм. Доступность того или иного варианта анализа определяется при настройке варианта анализа.

Пользователь может изменить:

  • Видимость варианта анализа в мониторе,
  • Вариант отображения (кратко, диаграмма, таблица),
  • Вывод деталей для краткого представления,
  • Вывод итогов для таблиц.
